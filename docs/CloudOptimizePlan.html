wait()

Waits for the market created by Plan#run to reach the "SUCCEEDED" state. This *could* result in an infinite loop, and doesn't have any provisions for a timeout. It also rapid-fire spams the API with no exponential backoff between checks. In short, it works, but it is heavy handed, and ripe for refactoring.
